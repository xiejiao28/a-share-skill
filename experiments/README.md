# experiments

`experiments/` 是这个仓库的策略研究工作区，用于放实验脚本、实验结果和实验报告。

目录约定：

- `experiments/scripts/`
  - 放实验脚本
  - 例如参数搜索、股票池验证、账户复盘辅助脚本
- `experiments/results/`
  - 放结构化实验结果
  - 优先保存为 JSON，便于后续脚本和 agent 继续读取
- `experiments/reports/`
  - 放结论优先的 Markdown 报告
  - 用于汇总实验结果、给出对比和建议

使用原则：

1. 先把原始结果写到 `results/`
2. 再基于结果生成 `reports/`
3. 不在报告里手工编造结果，报告应以 `results/` 为准
4. 新的实验优先复用已有脚本骨架，而不是重新写一套回测流程

当前这里已经承担两类工作：

- 交易策略参数调优
- 股票池固定参数验证

后续如果 agent 需要做策略研究、批量回测、结果对比或实验报告，优先从这个目录开始。
