# 优化 rsi_revert 参数

## 目标

在现有 `a-share-paper-trading` 回测引擎之上，为 `rsi_revert` 策略建立第一版单票参数调优实验，针对 `601669` 在近 3 年区间上搜索 `buy_rsi` / `sell_rsi` 参数组合，并按收益/回撤平衡输出结果排行。

## 已知信息

* 回测引擎已支持 `rsi_revert`
* 当前可调参数为：
  * `buy_rsi`
  * `sell_rsi`
* 父任务已确定第一阶段只做单票验证：
  * 股票：`601669`
  * 时间区间：近 3 年
* 调优结果排序原则不是单看收益，而是收益 / 回撤平衡

## 需求

* 在 `experiments/scripts/` 下增加 `rsi_revert` 参数调优实验脚本或相关实现
* 枚举 `buy_rsi` / `sell_rsi` 参数组合
* 保证参数约束：
  * `buy_rsi < sell_rsi`
* 调用现有回测引擎，而不是重复实现策略逻辑
* 将结果写入 `experiments/results/`

## 验收标准

* [ ] 能对 `601669` 跑出一批 `rsi_revert` 参数组合回测结果
* [ ] 输出结果包含收益、最大回撤、收益回撤比、交易次数等核心指标
* [ ] 排名体现收益/回撤平衡，而不是只看最高收益
* [ ] 结果文件成功落到 `experiments/results/`

## 明确不做

* 多股票批量验证
* 训练集/验证集拆分
* 图表化报告
