# 用最优参数测试股票池近一年表现

## 目标

基于 `experiments/reports/601669_strategy_comparison_2023-06-17_to_2026-06-17.md` 中已经识别出的最优参数，在指定股票池上测试近一年的收益率与最大回撤，并输出结果汇总。

## 已知信息

* 已有实验报告给出的当前最优参数为：
  * `sma_cross`: `fast=30`, `slow=60`
  * `rsi_revert`: `buy_rsi=25`, `sell_rsi=60`
* 当前仓库已经有可复用的实验脚本：
  * `experiments/scripts/optimize_backtest_params.py`
* 当前回测引擎已支持：
  * `sma_cross`
  * `rsi_revert`
* 用户指定的股票池为：
  * `600517` 国网英大
  * `601669` 中国电建
  * `000060` 中金岭南
  * `002041` 登海种业
  * `002140` 东华科技
  * `002546` 新联电子
  * `603105` 芯能科技
  * `000630` 铜陵有色
* 本轮测试周期要求为近一年。

## 临时假设

* 本任务优先复用现有实验脚本和回测引擎，不重新实现策略逻辑。
* 代码实现放在 `experiments/scripts/`。
* 结果会落到 `experiments/results/`，结论或摘要会落到 `experiments/reports/`。

## 需求

* 使用报告中已确定的最优参数，测试指定股票池近一年的表现。
* 同时跑两个策略：
  * `sma_cross`
  * `rsi_revert`
* 至少输出每只股票的：
  * 收益率
  * 最大回撤
* 结果应便于横向比较不同股票的表现。
* 结果摘要除逐股票明细外，还需要有按策略汇总的整体结论。
* 优先复用现有回测脚本与统一结果结构。

## 验收标准

* [ ] 指定股票池全部完成回测
* [ ] 每只股票至少有收益率和最大回撤结果
* [ ] 结果已落到 `experiments/results/`
* [ ] 有一份简要汇总结论落到 `experiments/reports/`
* [ ] 汇总结论同时包含逐股票明细和按策略汇总

* 训练集 / 验证集拆分
