# Relay Setups

## 高风险说明

本模块处理高风险接力场景。  
只有当市场环境允许、题材强度足够、承接和封板质量都在线时，才有讨论价值。默认仓位参考 `position-discipline.md` 中的 `light`。

## 核心场景

### 一进二

关注重点：

- 首板是否具备强度和辨识度
- 是否属于强题材而不是杂毛跟风
- 次日是否出现明确承接和延续意愿

trigger：

- 强题材 + 强首板 + 次日承接明确

invalidation：

- 次日明显弱承接
- 强度被同身位个股卡掉

### 二进三

关注重点：

- 二板是否具备带动性
- 前两板量能结构是否支持第三板延续
- 三板是否符合前缩后放、前放后缩的量价节奏

trigger：

- 题材仍强
- 竞价或盘中表现说明资金继续选择它做核心

invalidation：

- 三板阶段转为高开弱走
- 放量但封板不住，且题材后排明显掉队

### 2+2

关注重点：

- 两天连板后，两天断板整理是否仍由主导资金控制
- 整理阶段是否跌破核心参考位
- 是否具备再次启动条件

trigger：

- 整理未破关键支撑
- 再次走强时有承接和确认动作

invalidation：

- 整理阶段关键位失守
- 强度被新核心替代

## 共通判断项

### 强题材

- 题材本身仍在活跃
- 不是孤立个股硬冲

### 强卡位

- 同身位或同题材比较中占优
- 有辨识度，不是被动跟涨

### 强封板

- 封板动作坚决
- 封板后承接稳定

### 竞价承接

- 竞价表现与结构预期一致
- 没有明显弱承接或高开低走迹象

### 板块共振

- 个股强势最好有板块响应
- 若板块不跟，则需要显著降低预期

## 不及预期处理

以下情况都应视为接力逻辑转弱：

- 竞价承接明显不足
- 高开后无法继续转强
- 盘中接近封板却反复转弱
- 同题材后排快速掉队

一旦出现上述情形，应优先切换到 `exit-failure-rules.md`，而不是临时改成波段解释。

## 不适用场景

以下情况下默认不使用本模块：

- 环境为 `defensive` 或 `no_trade`
- 题材已明显退潮
- 个股已出现高位放量滞涨

## 输出格式

建议至少输出：

- 当前接力场景
- 题材强度判断
- 承接判断
- trigger
- invalidation
- risk
- position_grade
